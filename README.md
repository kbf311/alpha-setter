# Alpha Setter (画像半透明化・WebP変換ツール)

画像を指定した透明度（半透明など）にし、指定した背景色（白または黒）と合成した上で、WebP形式に一括変換するツールです。
元の画像ファイルを書き換えることなく、新しいフォルダへ安全に出力します。

---

## 主な機能

* **一括処理**: フォルダを選ぶだけで、その中にある複数の画像をまとめて処理できます。
* **背景色の選択**: 合成する背景色を「白」または「黒」から選べます。
* **透明度（アルファ値）の調整**: `0`（完全に透明）から `255`（完全に不透明）の間で、好みの不透明度を指定できます。（標準の半透明は `128`）
* **WebP形式で保存**: 画質をきれいに保ちつつ、ファイルサイズを軽量化できるWebP形式（.webp）で書き出します。
* **安全設計**: 元の画像ファイルは上書きせず、同じフォルダ内に作成される `convert` フォルダの中に保存されます。

---

## 準備（初回のみ）

本ツールを初めて使用する前に、一度だけ以下の準備を行ってください。

1. お使いのパソコンに **Python** がインストールされていることを確認してください。
2. フォルダ内にある **`run_setup.bat`** をダブルクリックして実行します。
3. 必要な準備が自動で行われます。黒い画面に「`セットアップ完了`」と表示されたら、何かキーを押して画面を閉じてください。

---

## 使い方

1. フォルダ内にある **`run_app.bat`** をダブルクリックして起動します。
2. **フォルダの選択**: 
   処理したい画像が保存されているフォルダを選択する画面が表示されます。対象のフォルダを選んで「フォルダーの選択」をクリックします。
3. **背景色の選択**: 
   「背景色は白でよろしいですか？」というメッセージが表示されます。
   * 白背景にする場合：**「はい」** をクリック
   * 黒背景にする場合：**「いいえ」** をクリック
4. **透明度の設定**: 
   透明度を入力する画面が表示されます。`0`〜`255` の数値を入力して「OK」をクリックします。
   * ※何も入力せずに「OK」を押すと、標準的な半透明（`128`）が適用されます。
5. **完了**: 
   処理が終わると「処理が完了しました」と結果が表示されます。「OK」を押して終了します。

---

## 処理された画像の保存先

選択したフォルダの中に、新しく **`convert`** というフォルダが作られます。
処理済みの画像（.webp）はこの `convert` フォルダの中に保存されます。
元の画像はそのまま残っていますのでご安心ください。

---

## 対応している画像形式

以下の拡張子の画像ファイルを自動で判別して処理します。
* `.png` / `.jpg` / `.jpeg` / `.bmp` / `.gif` / `.tiff` / `.webp`
